## üîÑ Migration Guide

### From v3.3.4 to v3.3.5

**Required Actions:**

1. **Upgrade package:**

   ```bash
   pip install --upgrade ign-lidar-hd
   ```

2. **Verify version:**

   ```bash
   ign-lidar-hd --version
   # Should show: ign-lidar-hd 3.3.5
   ```

3. **Or via Python:**

   ```python
   import ign_lidar
   print(ign_lidar.__version__)
   # Should show: 3.3.5
   ```

**None!** This release is 100% backward compatible with v3.3.4.
